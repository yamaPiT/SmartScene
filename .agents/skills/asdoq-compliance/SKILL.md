# ASDoQ文書品質モデルへの準拠 (ASDoQ Compliance)

本スキルの基準は `project-rules.mdc` のセクション3.4「ASDoQ 6大品質特性（軽量版）」として**常時適用**されています。

通常時はエージェントが内在化した6大品質特性に基づいて品質を担保します。
詳細な定義と基準に基づく重厚なレビューが必要な場合は、`docs/asdoq_model_markdown.md` を深く読み込んでください。
