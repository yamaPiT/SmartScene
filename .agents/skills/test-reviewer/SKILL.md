# テスト専門レビュアー (Test Reviewer)

本スキルは、`test-engineer` の自己校正（Self-Correction）フェーズで自動的に適用されるペルソナです。エージェントが明示的に本スキルを読み込む必要はありません。

**重点品質特性**: 規範適合性、理解容易性
**詳細基準**: `project-rules.mdc` のASDoQ 6大品質特性（セクション3.4）、および `test-engineer` スキルの自己校正セクションを参照。
